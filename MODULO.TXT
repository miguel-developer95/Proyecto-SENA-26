RF 1.1	El sistema permitirá registrar nuevos productos en el inventario con su información correspondiente.
RF 1.2	El sistema permitirá modificar la información de los productos registrados
RF 1.3	El sistema permitirá eliminar productos que ya no se encuentren disponibles
RF 1.4	El sistema permitirá visualizar el listado de productos y su cantidad disponible

RF 2.1	El sistema permitirá procesar una venta seleccionando productos mediante escaneo de código de barras o búsqueda manual.
RF 2.2	El sistema deberá actualizar automáticamente la cantidad de productos en el inventario al realizar una venta, descontando las unidades correspondientes y registrando el movimiento de salida asociado al ID de la venta, con el fin de garantizar la trazabilidad de la operación.
RF 2.3	El sistema permitirá registrar el pago en efectivo, transferencia digital (Nequi/Daviplata) o crédito ("fiar").
RF 2.4	El sistema permitirá pausar una venta en curso para atender a otro cliente y recuperarla después sin perder los datos cargados.
RF 2.5	El sistema permitirá cancelar una venta realizada, exigiendo un motivo y reintegrando los productos al stock.

RF 3.1	Registro de Compras: Captura de código, fecha, proveedor, productos, cantidades, precios, valor total, método de pago y responsable.
RF 3.2	Registro de Proveedores: Almacenar NIT, nombre, dirección, teléfono de contacto, correo, estado y fecha de registro.
RF 3.3	Historial de Compras: Consulta cronológica de compras con detalle de proveedor, productos y valores.
RF 3.4	Modificación de Compras: Mantener trazabilidad de cambios con fecha y responsable.

RF 4.1	El sistema generará informes que muestren la ganancia real (Precio Venta - Precio Compra) por producto o categoría.
RF 4.2	El sistema identificará los productos con mayor rotación (los más vendidos) y los que no se mueven.
RF 4.3	El sistema permitirá definir qué acciones puede hacer un "Vendedor" (solo ventas) frente a un "Administrador" (todo).
