RF 1.2	El sistema permitirá modificar la información de los productos registrados
RF 1.3	El sistema permitirá eliminar productos que ya no se encuentren disponibles
RF 1.4	El sistema permitirá visualizar el listado de productos y su cantidad disponible
RF 2.1	El sistema permitirá procesar una venta seleccionando productos mediante escaneo de código de barras o búsqueda manual.
RF 2.2	El sistema deberá actualizar automáticamente la cantidad de productos en el inventario al realizar una venta, descontando las unidades correspondientes y registrando el movimiento de salida asociado al ID de la venta, con el fin de garantizar la trazabilidad de la operación.
RF 2.3	El sistema permitirá registrar el pago en efectivo, transferencia digital (Nequi/Daviplata) o crédito ("fiar").
